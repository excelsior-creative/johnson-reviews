### Site redesign: review detail page (`/blog/[slug]`)

The single biggest visual gap vs. the reference. The previous template
was a generic centered `max-w-4xl` prose column with a rounded image —
zero editorial treatment. Now matches the reference's Gilded Legacy
review-detail page:

- New `ReviewDetailHero` — full-bleed image hero (80vh / 870px
  reference), grayscale + brightness dim, bottom-left eyebrow pills
  (category + location + date), giant Noto Serif title, italic gold
  tagline from `post.excerpt`.
- New `VerdictPanel` — ratings block (Food / Experience / Value /
  Kid-Friendly labels — family-travel adapted, NOT haute-cuisine).
  Gold-gradient filled bars on a `#4d4635` track. Renders nothing if
  no ratings data. Posts schema doesn't carry ratings yet; the panel
  is wired via loose optional field lookup so lighting it up is a
  zero-UI-change Payload schema addition whenever we're ready.
- New `ConciergeSidebar` — sticky card with Address / Price Tier /
  Hours / Category + gilded-gradient CTA to the venue's site. Same
  graceful-degradation pattern: renders nothing if no venue data.
- New `.review-prose` editorial body style in `globals.css`:
  Noto Serif, 1.75 leading, 65ch max-width, automatic gold drop-cap
  on the first paragraph, gold markers on lists, gold blockquote
  border.
- Page restructured into the reference's two-column layout (8-col
  narrative + 4-col sidebar) with inline 2-up staggered image
  fragment from the first two gallery images; remaining gallery images
